# dcard-clone with ptt-posts
Create a "Dcard" like webpage by React. 
And fill with ptt-posts that craw by backend server (maybe fastAPI). 
(only for my study use) 
 
### React structure
```
.
├── Root                   # Under App.js
│   ├── Top bar            # 上方的導航列, 搜尋列(要依照看板不同，搜尋不同版, 登入登出狀態)
│   ├── Body               # 中間區塊
│   │   ├── Left panel     # 看板選區
│   │   └── Middle block   # 文章清單
│   └── Other
└── README.md
```
